# Smart Interview Scheduler - Backend

## 📖 Overview

The **Smart Interview Scheduler Backend** is a RESTful web service developed using **Spring Boot** and **PostgreSQL**. It simplifies the interview scheduling process by providing APIs to manage candidates, interviewers, interview slots, and interviews. The application follows a clean layered architecture, ensuring scalability, maintainability, and efficient database management.

---

## 🚀 Features

- Candidate Management APIs
- Interviewer Management APIs
- Interview Scheduling APIs
- Interview Slot Management APIs
- CRUD Operations (Create, Read, Update, Delete)
- Global Exception Handling
- PostgreSQL Database Integration
- Layered Architecture (Controller → Service → Repository)
- RESTful API Development
